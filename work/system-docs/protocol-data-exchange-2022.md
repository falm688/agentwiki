# 开放式基金业务数据交换协议（2.2版）

**一句话结论：基金行业最核心的数据交换标准协议，定义了开户、交易、分红等全业务流程的数据格式和编码规范，是TA系统与销售渠道、托管行之间数据交互的基础。**

## 数据处理规则

### 数据类型定义

| 类型 | 说明 |
|:---|:---|
| C | 字符型 |
| A | 字符型，填数字 |
| N | 数字型 |

### 处理规则

1. 数字左补零右对齐，字符右补空格左对齐
2. 字符不区分大小写
3. 文本文件定长记录方式
4. 每行一条完整记录
5. 换行：回车(0DH) + 换行(0AH)
6. 小数点不传

## 关键要点

- 这是基金行业最基础的数据交换协议
- 所有TA系统、销售系统、托管系统都需遵循
- 业务编码是三位数字，申请类以0开头，确认类以1开头
- 协议持续演进，从2011版→2018版→2021版→最新2.2版
