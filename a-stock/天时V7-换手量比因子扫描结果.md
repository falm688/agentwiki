## **TR>2（日换手率>2%）是最佳新增因子：年化+19.4%，回撤仅-22.7%，大幅改善V6基线**

### 扫描概述

- **日期**: 2026-06-04
- **引擎**: strategies/core/backtest.py（参数化引擎，全共享数据）
- **数据**: daily_basic 通过 sync_factors.py 补全所有字段 (100%覆盖率)
- **参数**: TP=16% / SL=12% / ED=8% / HD=25 / MX=3 / 1%卖滑点
- **回测期**: 2021-01-04 ~ 2026-05-26 (1547天)
- **扫描组合**: 18组（换手率 + 量比 + MV组合）

### 关键发现

1. **TR>2（日换手率>2%）是最佳因子**：年化+19.4%，回撤仅-22.7%。相比V6基线：
   - 年化提升 +5.0pp（+14.4% → +19.4%）
   - 回撤减少 -14pp（-36.6% → -22.7%）
   - 胜率提升 +2.3pp
   - 交易次数几乎不变（241→240）

2. **TR>5（换手率>5%）收益更高但风险也大**：年化+23.7%但回撤-36.5%，波动太大。

3. **量比(VR)过滤效果中规中矩**：VR>1年化+16.9%/回撤-33.7%，略优于基线但不如TR>2。

4. **MV+组合全部失效**：MV 20-200亿 + 换手率/量比的双重过滤后，符合条件的候选股为0。说明MV 20-200亿的小盘范围与高换手条件不重叠——小盘换手率通常不高。

### 结论与后续

- **候选新基线**: V6 + TR>2（年化+19.4%, 回撤-22.7%）
- **需交叉验证**: TR>2可能对2021年初高换手妖股行情有特殊适应性，需切时间段验证
- **下一步**: 将TR>2与之前MV 20-200亿因子合并测试（顺序无关，取并集？）
- **风险提示**: TR>5 +23.7%年化需要怀疑是否过拟合换手率阈值
