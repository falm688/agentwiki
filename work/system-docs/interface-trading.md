# TA 与投资交易系统接口规范 V3.3

**一句话结论：TA系统向投资交易系统导出资产变动数据，以CSV格式文本文件传输，涵盖申购、赎回、分红、强增、强减等业务类型。**

## 接口定义

### 文件格式

- **文件名**: `AssetchgYYYYMMDD`
- **格式**: 文本文件，CSV格式（逗号分隔）
- **每行**: 一个基金的一条数据

### 业务类型

| 代码 | 业务类型 | 说明 |
|:---|:---|:---|
| 02 | 申购 | 基金份额增加 |
| 03 | 赎回 | 基金份额减少 |
| 50 | 基金成立 | 基金份额增加 |
| 13 | 基金转换出 | 基金份额减少 |
| 16 | 基金转换入 | 基金份额增加 |
| 74 | 现金分红 | 只包含现金分红金额 |
| 73 | 再投资1 | 只包含再投资金额 |
| 75 | 再投资2 | 红利再投资份额 |
| 70 | 强增 | 基金份额增加 |
| 71 | 强减 | 基金份额减少 |
| PF | 保本金额 | 中登TA保本基金（不区分销售机构） |

### 数据字段

| 字段 | 类型 | 说明 |
|:---|:---|:---|
| 业务类型 | char(2) | 见上表 |
| 基金代码 | char(6) | - |
| 交易申请日期 | char(8) | 上一个工作日，YYYYMMDD |
| 成交份额 | decimal(16,2) | 申购/成立/转换入/再投资/强增=增加；赎回/转换出/分红/强减=减少 |
| 成交金额 | decimal(16,2) | 基金资产变动金额（不含手续费） |
| 手续费 | decimal(16,2) | 所有税费 |
| 过户费 | decimal(16,2) | 专户产品赎回时为业绩报酬 |
| 印花税 | decimal(16,2) | 分红时为所得税；专户产品分红时为业绩报酬 |
| 后收费用 | decimal(16,2) | 赎回后收认/申购费、基金转换补差费 |
| 其他费用 | decimal(16,2) | 专户产品赎回时为违约退出费 |
| 归基金资产费用 | decimal(16,2) | 交易费中归基金资产的部分 |
| 交收日期 | char(8) | 资金划账日期 |
| 处理日期 | char(8) | 当前日期 |
| 单位分红 | decimal(16,4) | 仅分红业务有效 |
| 销售机构 | char(3) | 销售商代码（可能含字母） |
| 结算币种 | char(3) | 156:人民币; 344:港币; 840:美元 |

### 数据唯一性

每天的数据中，**业务类型 + 基金代码 + 销售机构** 三个字段唯一。

## 专户产品特殊处理

| 字段 | 普通基金 | 专户产品 |
|:---|:---|:---|
| 过户费 | 过户费 | 赎回时=业绩报酬 |
| 印花税 | 印花税 | 分红时=业绩报酬 |
| 其他费用 | - | 赎回时=违约退出费 |
| 归基金资产费用 | 交易费归基金部分 | 赎回时含违约退出费 |

## 分红业务特殊处理

- **74 现金分红**: 只包含现金分红金额
- **73 再投资1**: 只包含再投资金额
- **75 再投资2**: 红利再投资份额
- 73 和 74 的交收日期相同

## 关键要点

- CSV格式，逗号分隔
- 日期格式：YYYYMMDD
- 专户产品有多处字段复用（业绩报酬、违约退出费）
- 分红业务拆分为三条记录（74/73/75）
