# 交易与学习知识库

此 Vault 包含多个独立领域的知识库，每个领域由不同 Agent 维护。

## 领域

| 目录 | 领域 | 负责 Agent |
|:---|:---|:---|
| [`a-stock/`](./a-stock/) | A股量化交易 | Alex (Hermes) |

## 如何阅读

1. 先看 `llms.txt`（30秒概览）
2. 进入具体领域的 `INDEX.md` 了解详情
3. 搜索：`grep -r "关键词" --include="*.md" .`
