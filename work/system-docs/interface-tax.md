# 登记过户系统与客户服务系统接口规范 — 非居民金融账户涉税信息接口

**一句话结论：TA系统向客服系统报送CRS（共同申报准则）非居民金融账户涉税信息，以文本文件定长记录方式传输，覆盖新开账户和存量账户的尽职调查。**

## 版本信息

- **来源文件**: 登记过户系统与客户服务系统接口规范_非居民金融账户涉税信息接口.doc
- **文件大小**: 141,312 bytes
- **新增日期**: 2017-12-29
- **更新日期**: 2020-05-26（调整接口字段描述）

## 数据处理规则

与客服系统接口规范一致：
- 文本文件定长记录
- 新增字段增加在尾部，客服系统不校验记录长度
- 第一行: 数据记录数
- 最后一行: `END`
- 换行: 回车(0DH) + 换行(0AH)
- 数字左补零右对齐，字符右补空格左对齐
- 小数点不传，默认2位小数
- 负数第一位用 `-` 表示

## 更新规则

- **频率**: 每日一次
- **范围**: 截至当日做完登记过户操作后的投资人信息
- **模式**: 每次只提供当天报送（新增/更新）的全部数据及结果

## 地址格式规范

**LivingAddress2（中文固定格式）**:
- 格式: 省\|市\|县\|邮编
- 省、市：必填，填写民政部最新行政区划代码（六位数字）

**LivingAddress4（英文固定格式）**:
- 格式: 城市\|街道\|楼号\|房门号\|楼层\|区\|邮箱\|邮编\|行政区划
- 城市：必填
- 境内地址：城市/行政区划填写拼音
- 境外地址：填写英文

## 关键要点

- 单名投资者：英文姓填单名，英文名填 `NFN`
- 只有中文名的：英文姓/名填写拼音
- 机构投资者的现居地址字段应同属办公地址或注册地址
- 新开户尚未获得基金账号时，TAAccountID 可填空，但须填报原申请日期和原申请单编号
