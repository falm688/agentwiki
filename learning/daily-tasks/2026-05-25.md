### 2. 类型系统学习（20min）

**核心概念：**
- Python 是动态类型：变量类型在运行时确定
- Java 是静态类型：变量类型在编译时确定
- `type()` 查看类型，`isinstance()` 判断类型

**对比 Java：**

| 特性 | Python | Java |
|:---|:---|:---|
| 类型声明 | 不需要 | 必须 |
| 类型检查 | 运行时 | 编译时 |
| 变量复用 | 可以改类型 | 不能改 |

**练习：**
```python
# 观察动态类型
x = 10        # int
x = "hello"   # str，不报错
print(type(x))  # <class 'str'>

# isinstance 判断
print(isinstance(x, str))  # True
```

### 3. f-string 格式化（20min）

**语法：** `f"...{表达式}..."`

**金融数据格式化练习：**
```python
stock = {"code": "600519", "name": "贵州茅台", "price": 1680.0, "change": 2.3}

# 基础格式化
print(f"{stock['name']} 当前价格: {stock['price']}")

# 对齐 + 精度
print(f"{'名称':<10} {'价格':>8} {'涨跌幅':>8}")
print(f"{stock['name']:<10} {stock['price']:>8.2f} {stock['change']:>+7.2f}%")

# 千分位分隔
large_num = 12345678.9
print(f"市值: {large_num:,.2f}")  # 12,345,678.90
```

**输出目标：**
```
贵州茅台 当前价格: 1680.0
名称        价格   涨跌幅
贵州茅台   1680.00   +2.30%
市值: 12,345,678.90
```
