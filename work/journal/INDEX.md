# 每日工作流水索引

**一句话结论：按日期记录每天的工作内容，自动提取对话中的工作事项。**

## 记录规则

- 每次对话中涉及的工作内容自动追加到当日文件
- 格式：时间戳 + 任务 + 结果 + 关键信息
- 私人对话说"不记"或"私事"时跳过

## 记录格式

```markdown
## HH:MM — 任务简述

- **任务**: 具体做什么
- **结果**: 成功/失败/进行中
- **关键信息**: 代码、决策、问题
- **关联**: [相关页面](../insights/xxx.md)
```
