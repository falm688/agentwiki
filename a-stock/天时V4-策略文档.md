# 天时V4 — 气候分类器 + V4超跌反转 + ETF轮动

## 一句话结论

**天时V4（vol20_low=20，SL=12%，ret5企稳过滤）在2022~2026熊牛完整周期中取得 +258.8%/年化33.7%/回撤21.9%/胜率61%，大幅超越纯V4潜龙。** ✅

## 策略架构

```
气候分类器V3.1
    │
    ├─ V4模式 (56%)  → 超跌反转选股：60日跌>8%→近5日不再暴跌→买3只→TP=12%/SL=12%/25天出场
    ├─ TREND模式 (22%) → ETF动量轮动：4指数(HS300/中证500/创业板/上证50)20日动量选最强 
    └─ CASH模式 (22%) → 空仓（持有V4仓位不强制平仓，仅停新买）
```

### V3.1分类器关键改进

1. **熊市CASH保护**：高/中波动区 + below MA60 + ret60<-8% + 5日无反弹(2%) → CASH
2. **ret5秒切出**：短时反弹 >2% → 立即切回V4，不等到ret60恢复（解决了COVID误判）
3. **不强制平仓**：CASH/TREND时不卖掉V4持仓，让它们自然TP/SL/HD出场
4. **入场企稳过滤**（V4最终版新增）：`ret5>=-0.05` — 只买近5日不再暴跌的股票，避免接飞刀

### 入场企稳过滤原理

传统V4只检查"60日跌>8%"（跌得多=便宜），但没检查"还在不在跌"。加入 `ret5>=-0.05` 后：
- **拦截**：近5日暴跌超过5%的股票不买（还在加速下跌，别接飞刀）
- **允许**：近5日跌幅<5%甚至企稳/反弹的才买（跌到位了，等反弹）
- 效果：SL止损从50笔降到37笔（-26%），胜率从56%升到61%

## 核心原理

天时V4的V3.1气候分类器核心设计：

- **vol20_low=20**：只有vol20>20%才进入中/高波区，此时如果市场在MA60下方且有持续下跌（ret60<-8%且无反弹），判CASH
- **ret5反弹检测**：解决了ret60滞后问题——跌完了但ret60还显示负数时，只要短时反弹>2%就切回V4
- **不强制平仓**：CASH/TREND时保留V4持仓，避免模式切换带来的摩擦成本
- **企稳过滤**：V4入场时检查ret5>=-0.05，只买"跌够了"的股票
