# TA 与会计清算接口规范 V2.66

**一句话结论：TA系统向会计/托管系统导出资金清算数据，涵盖净值、交易、分红、份额调整、转托管等多类文件，支持前后收费模式及专户产品。**

## 接口文件清单

| 文件类型 | 说明 |
|:---|:---|
| NAV | 基金净值文件 |
| Income | 基金收益情况 |
| Subscribe | 认购数据 |
| Purchase | 申购数据 |
| Redeem | 赎回数据 |
| SwitchOut | 基金转换出 |
| Switch | 基金转换入 |
| Bonus | 分红数据 |
| AdjustShare | 份额调整 |
| ChangeAgency | 转托管数据 |
| Frozen | 冻结资金 |

## 关键字段

### 份额类别
- V1.x 版本：无份额类别
- V2.x 版本：增加份额类别字段
  - `A`: 前收费
  - `B`: 后收费
  - 后续可能有 C、D 等

### 文件名约定

给托管行的文件，文件名后增加 `_AAA_BB`：
- `AAA`: 托管行代码
- `BB`: TA代码

## 专户产品特殊处理

| 字段 | 普通基金 | 专户产品 |
|:---|:---|:---|
| 其它费用 | 预留 | 赎回时=违约赎回费 |
| 归基金资产费用 | 交易费归基金部分 | 赎回时含违约赎回费 |
| 业绩报酬 | - | 2011年后独立字段 |

## 关键要点

- 版本号大于 V2.0 表示包含份额类别字段
- 同时提供两个版本（如 V1.4 和 V2.4）以保持兼容
- 货币基金有特殊字段（万份收益、年收益率等）
- QDII基金支持外币计价净值
