## 项目 0：股票筛选器 CLI

### 需求

用纯 Python（无 Pandas）实现一个命令行股票筛选工具：

1. **输入**：股票列表（代码、名称、价格、涨跌幅）
2. **筛选条件**：
   - 涨幅 > 5% 的股票
   - 价格 < 50 元的股票
   - 用列表推导式实现
3. **输出**：格式化表格（f-string）
4. **扩展**：
   - 支持从 CSV 文件读取
   - 支持多种筛选条件组合

### 技术约束

- ❌ 不允许用 Pandas（P0 阶段用基础数据结构）
- ✅ 必须用列表推导式
- ✅ 必须用 f-string 格式化
- ✅ 代码上传到 GitHub

### 示例输出

```
股票筛选结果
============
代码    名称        价格    涨跌幅
600519  贵州茅台    1680.0  +2.3%
000858  五粮液      145.5   +1.8%

共筛选出 2 只股票
```
