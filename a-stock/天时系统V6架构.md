# 天时系统 V6 架构总览

> **一句话结论：天时V6是一个"V4超跌反转为主、ETF动量轮动为辅、气候分类器自动切换"的全自动量化交易系统，当前年化+20.7%/回撤-22.7%（仅V4模式），TREND模式的ETF轮动待集成。**

```ascii
╔══════════════════════════════════════════════════════════════════╗
║                   天时系统 V6 架构总览                           ║
╚══════════════════════════════════════════════════════════════════╝

┌─ 数据层 ───────────────────────────────────────────────────────┐
│  core/data.py  →  DataLoader                                │
│     - MySQL连接 (qteasy_db, 8.5M行stock_daily + 5.9M行mf)    │
│     - 前复权价格缓存 (O(1)字典查询)                            │
│     - 股票池 (3011只, 排除ST/创业板/科创板)                     │
│     - 因子加载 (FactorStore, 时间感知)                         │
│     - HS300 + ETF指数数据                                      │
└──────────────────────────────────────────────────────────────┘

┌─ 核心引擎 ──────────────────────────────────────────────────────┐
│                                                              │
│  climate_classifier_v3.py → ClimateClassifierV3               │
│  ┌─────────────────────────────────────────────────┐         │
│  │ 输入: HS300价格(Ndarray) + 日期索引              │         │
│  │ 输出: 3种模式 (V4 / TREND / CASH)               │         │
│  │ 判定依据: vol20 + above_ma60 + ret60 + ret5     │         │
│  │ 参数: vol20_low=20 / v20h=25 / v20e=35          │         │
│  │             bear_ret60=-0.08                     │         │
│  └─────────────────────────────────────────────────┘         │
│                                                              │
│  core/backtest.py → run(params, dl)                         │
│  ┌─────────────────────────────────────────────────┐         │
│  │ 回测主循环: 每日循环 →                         │         │
│  │  ① 气候分类器判定当前模式                       │         │
│  │  ② 管理持仓(TP/SL/HD出场)                       │         │
│  │  ③ 非CASH模式下V4选股买入                      │         │
│  │     - 60日跌超ED% (8%)                          │         │
│  │     - STAB企稳过滤 (5日不继续跌)                │         │
│  │     - 因子过滤 (TR>2, 可选)                     │         │
│  │  ④ TURN模式下ETF轮动选ETF                       │         │
│  │  ⑤ 净值计算                                    │         │
│  └─────────────────────────────────────────────────┘         │
│                                                              │
│  默认参数:                                                 │
│    ED=8% | TP=16% | SL=12% | HD=25天 | MAX=3只            │
│    v20l=20 | br=-0.08 | stab=-0.05 | sllp=1%              │
│    因子: TR>2 (换手率>2%)                                   │
│                                                              │
└──────────────────────────────────────────────────────────────┘

┌─ 策略模式 ─────────────────────────────────────────────────────┐
│                                                              │
│  气候分类器                                                       │
│      │                                                      │
│      ├── V4模式 (57%时间)                                    │
│      │   └─ 个股超跌反转选股                                  │
│      │      - 60日跌>8% → 超跌候选                          │
│      │      - 按跌幅排序取前3名                              │
│      │      - TP=16%止盈 / SL=12%止损 / HD=25天到期          │
│      │      - TR>2过滤（高换手活性股）                      │
│      │      - 2021-2026年化:   +20.7% / 回撤 -22.7%  ✅      │
│      │                                                      │
│      ├── TREND模式 (24%时间) [ETF动量轮动待集成]             │
│      │   └─ 4宽基指数动量轮动  ← 当前开发中                  │
│      │      - 沪深300/中证500/创业板/上证50                 │
│      │      - 20日动量最强+MA60过滤 → 全仓1只              │
│      │      - 换仓阈值5%，最少持仓8天                        │
│      │      - 2021-2026年化: +12.2% / 回撤 -0.2%   已验证    │
│      │                                                      │
│      └── CASH模式 (19%时间)                                  │
│          └─ 空仓(不强制平仓V4持仓)                           │
│             - 仅禁止新买入，已有V4持仓自然走到TP/SL/HD        │
│                                                              │
│  当前V6+TR2 = 只有V4模式 + TR>2过滤（分类器未启用）           │
│  因为TREND模式没有替代策略，分类器开启的话TREND期会空仓        │
│                                                              │
└──────────────────────────────────────────────────────────────┘

┌─ 模拟盘系统 ────────────────────────────────────────────────────┐
│  core/simulator.py → Simulator('策略名')                      │
│                                                              │
│  ┌─────────────────────────────────────────────────┐         │
│  │ 每工作日 18:30 CST 自动执行:                     │         │
│  │ ① DataLoader加载最新数据                        │         │
│  │ ② 风控出场(TP/SL/HD)                           │         │
│  │ ③ V4选股(V4 filter + TR>2)                     │         │
│  │ ④ 买入(96%资金等权3只)                          │         │
│  │ ⑤ 保存状态到 sim_{策略名}_state.json             │         │
│  │ ⑥ 飞书推送日报                                 │         │
│  └─────────────────────────────────────────────────┘         │
│                                                              │
│  运行中策略:                                                 │
│  ✅ V6+TR2  (v6tr2_daily_signal.py + cron)                   │
│    当前净值: ¥96,718 (-3.3%)  持仓: 3只                      │
│                                                              │
│  每周复盘 (周五18:30):                                        │
│  ✅ v6tr2_weekly_review.py + cron                            │
│                                                              │
└──────────────────────────────────────────────────────────────┘

┌─ 每日数据流 ───────────────────────────────────────────────────┐
│                                                              │
│   14:50 尾盘                                            │
│     ↓                                              │
│   MySQL读取当日数据                                    │
│     ↓                                              │
│   V4选股 (60日跌幅排序 + TR>2过滤 → Top3)          │
│     ↓                                              │
│   Simulator执行                                     │
│     - 卖出: TP/SL/HD持仓                              │
│     - 买入: 候选股票(尾盘价≈收盘价)                     │
│     ↓                                              │
│   更新 sim_v6tr2_state.json                          │
│     ↓                                              │
│   飞书推送: 今日操作 + 持仓 + 净值                   │
│                                                              │
└──────────────────────────────────────────────────────────────┘

┌─ 数据来源 ──────────────────────────────────────────────────────┐
│  MySQL (qteasy_db, tushare数据源)                              │
│  ├─ stock_daily      (8.2M行) 日线行情 + 前复权计算            │
│  ├─ index_daily      (4指数)  HS300/中证500/创业板/上证50       │
│  ├─ daily_basic      (因子)  turnover_rate, circ_mv等          │
│  ├─ moneyflow_factors (5.9M行) 资金流: net_big_money等         │
│  └─ stock_basic      (股票池)                                  │
│                                                              │
│  腾讯实时行情API (qt.gtimg.cn) — 模拟盘实时价格查询             │
│  Tushare MCP (2000积分) — 补充数据/龙虎榜/财务因子             │
└──────────────────────────────────────────────────────────────┘

┌─ 文件结构 ──────────────────────────────────────────────────────┐
│  strategies/                                                    │
│  ├── core/                    核心模块 (7个)                    │
│  │   ├── data.py              DataLoader 数据层                │
│  │   ├── backtest.py          参数化回测引擎 V6/V7             │
│  │   ├── factors.py           FactorStore 因子管理              │
│  │   ├── simulator.py         多策略模拟盘引擎                  │
│  │   ├── realtime.py          腾讯实时行情API                  │
│  │   ├── trend.py             右侧趋势指标                      │
│  │   └── audit.py             审计数据包编译器                  │
│  ├── scripts/                 回测/扫描脚本                    │
│  ├── tools/[active/]          运行中工具                       │
│  │   ├── v6tr2_daily_signal.py  V6+TR2每日模拟盘              │
│  │   ├── v6tr2_weekly_review.py 每周复盘                      │
│  │   └── v6tr2_daily_wrapper.sh cron wrapper                  │
│  ├── active/                  历史回测脚本存档                  │
│  ├── archive/                 废弃方向                          │
│  ├── goldminer/               掘金量化版                       │
│  └── output/                  模拟盘状态文件                    │
│      ├── sim_v6tr2_state.json                                 │
│      ├── sim_v6tr2_daily.json                                 │
│      └── sim_v6tr2_weekly.json                                │
│                                                              │
│  climate_classifier_v3.py  分类器 (strategies/根目录)           │
└──────────────────────────────────────────────────────────────┘
```

## 当前状态

| 组件 | 状态 | 说明 |
|:----|:---:|:----|
| V4超跌反转 (个股) | ✅ 运行中 | 年化+20.7%/回撤-22.7% |
| TR>2换手率过滤 | ✅ 已集成 | 有效提升胜率从49%→52% |
| 气候分类器V3.1 | ✅ 就绪 | 3模式(V4/TREND/CASH) |
| ETF动量轮动 | 🟡 待集成 | 已验证+12.2%/回撤-0.2% |
| TREND+ETF轮动 | 🟡 待开发 | 气候分类器启用的最后拼图 |
| CASH空仓保护 | 🟡 待启用 | 需TREND集成后一起部署 |
| A杀二波战法 | ❌ 存档 | 19,039事件分析：43%随机性，无有效因子 |
| 资金流因子 | ❌ 无效 | 6M行数据，与V6基线完全一致 |
| 财务因子 (ROE/EPS) | ❌ 无效 | 与V6基线无区别 |

## 历史验证路径

```
潜龙(纯V4, SL=12%):                           +179%/17.4%   ← 起点
  ↓ +气候分类器V2
天时V2(SL=12%):                               +194%/18.4%
  ↓ +V3熊市CASH(ret5反弹)
天时V3.1(SL=12%, vol18):                      +220%/20.0%
  ↓ +vol20_low=20参数优化
天时V4(vol20_low=20):                         +235%/20.8%
  ↓ +TR>2过滤因子
天时V6(V4+TR2, TP=16%):                       +20.7%/-22.7% ← 当前最优
  ↓ +ETF动量轮动(TREND)+CASH(全模式)
天时V7(V4+ETF+CASH):                          待验证
```
