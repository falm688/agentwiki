# 双策略实盘操作手册

> 版本: v1.0 | 适用资金: 10万 | 更新: 2026-05-25

---

## 一、每日流程

### 收盘后（15:00-18:30）

监控自动运行，无事可做。

### 开盘前（9:25-9:30）

查看监控报告：

```bash
cat ~/trading_monitor.log | tail -3
```

或用飞书查看监控推送。

### 四种可能的结果

| 报告内容 | 操作 |
|:--------|:----|
| `"建议: 无需操作"` | 关掉，今天不交易 |
| `"建议: V4买入/卖出"` | 打开APP执行 |
| `"建议: 右侧调仓"` | 打开APP执行（月底） |
| `"⚠️ 策略切换: XXX→XXX"` | 按指引操作 |

## 二、V4 模式操作

> 适用场景: MIXED（结构性震荡）

### 日常管理

V4 最多持有 **2只**，每只 **5万**（50%仓位）

**买入信号**（监控输出）：
```
V4买入: 000XXX.SZ, 买入5000元
```
→ 9:30 开盘价买入 **5,000元**
→ 输入证券代码 → 买入 → 市价/现价

**卖出信号**（触发止盈/止损/持仓到期）：
```
V4卖出: 000XXX.SZ, 止盈6%触发
```
→ 9:30 开盘价卖出全部持仓

### 止盈止损规则

| 条件 | 动作 | 说明 |
|:----|:----|:----|
| 盈利 ≥ 6% | 卖出 | 次日开盘执行 |
| 亏损 ≥ 10% | 卖出 | 次日开盘执行 |
| 持有 ≥ 15天 | 卖出 | 次日开盘执行 |
| 市场转 BEAR | 清仓 | 全部卖出 |

### 新入场条件

- 持仓 < 2只
- 市场状态为 MIXED
- 有新的超跌候选（120日跌幅>8%，价格<MA60的95%）

### 止损不必设条件单

因为 V4 的止损是**收盘后判断、次日开盘执行**。10% 的止损幅度已经足够覆盖隔夜风险。如果某天直接跌停 10%，第二天开盘卖出即可——共亏约 10-20%，可控。

## 三、右侧模式操作

> 适用场景: BULL（牛市/赚钱效应好）

### 月频调仓

**每月末最后一个交易日**

监控输出：
```
右侧调仓: 卖出旧10只 → 买入新10只
新持仓: 000XXX(5%), 001XXX(5%), ... (共10只)
```

**次月第一个交易日开盘**

操作步骤：
1. 卖出当前持仓中的全部股票（市价卖出）
2. 用全部资金买入新清单的10只（每只约10%，即约1万/只）

耗时约 **5分钟**。

### 月内管理

- **不操作**，除非触发策略切换
- 即使期间某只大涨/大跌，也等到月末调仓再处理

### 最大回撤

右侧策略在牛市期间最大回撤约 **15%**（2025年实盘回测数据）。如果账户浮亏超过 20%，检查是否市场状态已切换——不要恐慌卖出。

## 四、策略切换流程

### 状态变化及应对

```
当前→目标    触发条件                   操作
────────────────────────────────────────────────
BULL→MIXED  HS300跌破MA60             卖出右侧 → 切换V4
            赚钱效应消失(中位数转负)

MIXED→BULL  HS300站回MA60             卖出V4 → 切换右侧
            赚钱效应恢复

*→BEAR      HS300跌超10%+6成票下跌     全部清仓 → 现金
             
BEAR→*      市场恢复                   恢复正常策略
```

### 切换操作示例

**场景：BULL→MIXED**
```
监控提示:
  ⚠️ 市场状态切换: BULL → MIXED
  ⚠️ 建议策略切换: 右侧 → V4

开盘操作:
  1. 卖出所有10只右侧持仓（市价）
  2. 资金转回活期/货币基金
  3. 查看V4信号，有候选则逐步建仓（每次1-2只）
```

切换时**先卖后买**，避免满仓踏空。

## 五、资金管理

| 状态 | 仓位 | 说明 |
|:----|:---:|:----|
| BULL | 满仓100% | 10只×10% |
| MIXED | 10%-50% | 1-2只×5% |
| BEAR | 0% | 现金/货币基金 |

### 加仓节奏

V4 模式下，每天最多新开 **1只**，避免一天内全部买在高点。

### 资金闲置处理

BEAR 和 MIXED 期间的部分现金：
- 放券商活期理财（如华宝添益 511990）
- 或货币基金
- 不要买任何其他产品

## 六、风险边界

| 风险 | 上限 | 处理 |
|:----|:---:|:----|
| 单笔最大亏损 | 10% | 止损触发 |
| 单日最大亏损 | 20%（双止损） | 概率极低 |
| 连续错误切换 | 最多3次 | 暂停交易1周 |
| 总回撤警戒线 | 30% | 停止所有策略，复盘 |

### 暂停条件

以下任一发生，**暂停交易一周**：
- 一个月内策略切换 ≥ 3次
- 账户回撤 ≥ 30%
- 连续 3 次操作信号被验证为错误

暂停期间：
1. 监控继续运行
2. 不执行任何买入
3. 一周后复盘决定是否恢复

## 七、实操清单

### 每天检查

- [ ] 监控正常输出（看日志时间戳）
- [ ] 有建议操作 → 记下来明早执行
- [ ] 无需操作 → 关掉

### 每次操作

- [ ] 开盘前看到信号（9:25-9:30）
- [ ] 执行交易（9:30-9:35）
- [ ] 核对成交（9:35-9:40）
- [ ] 确认持仓数量和预期一致

### 每月

- [ ] 月末检查右侧调仓信号
- [ ] 汇总当月收益
- [ ] 检查是否需要调整参数

## 八、启动步骤

资金解封后：

```
第一步: 配置Tushare Token（已配置）
第二步: 确认每日监控正常运行（✅ 已设置cron）
第三步: 第一天 → 查监控 → 看市场状态
第四步: 按状态进入对应策略
第五步: 逐步建仓（不要一天全买满）
```

**特别提醒**：第一天不要全仓买入。即使是 BULL，也分 2-3 天建仓，避免买在短期高点。
