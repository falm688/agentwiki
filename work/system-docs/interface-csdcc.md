# 中国结算开放式基金登记结算系统管理人数据接口规范

## 关键要点

- 中国结算（CSDCC）是基金行业的中央证券存管机构
- 该规范是管理人系统与中登系统对接的法定标准
- 业务编码与开放式基金业务数据交换协议基本一致
- 申请类编码以 0 开头，确认类编码以 1 开头
- 包含ETF特殊业务（一次/二次申购赎回）
- 包含创新业务（定期定额、强增强减、基金拆分合并等）
