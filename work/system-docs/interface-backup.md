# 基金行业数据集中备份接口规范 V2.2

**一句话结论：基金行业（含公募、私募）数据集中备份规范，定义了TXT格式定长记录的报送标准，涵盖账户、交易、份额、分红等全量业务数据。**

## 版本信息

- **版本**: V2.2
- **发布日期**: 2023-03-09
- **来源文件**: 基金行业数据集中备份接口规范V2.2-20230309.docx
- **适用对象**: 基金TA、管理人及其他相关参与人

## 核心目标

实现基金行业数据的集中备份，促进基金业整体数据安全与行业发展。

## 术语定义

### 基金业务当事人

| 角色 | 定义 |
|:---|:---|
| 基金管理人 | 依法从事投资基金管理的基金管理公司 |
| 基金托管人 | 依法保管基金资产、监督管理人投资活动的机构 |
| 注册（过户）登记人（TA） | 负责投资人账户保管、交易记录保存、代理分红、投资人账户报告、登记基金份额等服务的机构 |
| 基金持有人 | 持有证券投资基金份额的投资人 |
| 基金销售人 | 销售基金的法人，包括基金管理人（直销商）及经中国证监会注册的其他机构（代销商） |

### 业务种类

| 业务 | 定义 |
|:---|:---|
| 开户 | 投资人申请设立基金数量账户和资金账户 |
| 认购 | 募集期间申请购买基金 |
| 申购 | 开放期内申请购买基金份额 |
| 赎回 | 将基金份额兑换为现金 |
| 巨额赎回 | 单个开放日净赎回申请超过总份额10% |
| 预约赎回 | 提前发出未来某日的赎回申请 |
| 分红 | 将基金运作收益按比例分配给持有人 |
| 非交易过户 | 赠与、继承、司法判决等涉及的过户 |
| 基金转换 | 同一管理人管理的不同基金之间的转换 |

## 数据处理规则

1. **数字**: 左补零右对齐
2. **字符**: 右补空格左对齐，不区分大小写
3. **文件格式**: 文本文件定长记录方式
4. **记录格式**: 每行一条完整记录
5. **换行符**: 回车(0DH) + 换行(0AH)
6. **小数处理**: 数值型数据不传小数点

对通过公网传输的敏感数据强烈建议进行加密处理。
